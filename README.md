* Step 1 = [22, 27, 16, 2, 18, 6]
* Step 2 = [2, 22, 27, 16, 18, 6]
* Step 3 = [2, 6, 22, 27, 16, 18]

> Big-O gösterimini yazınız.

Insertion Sort’un time complexity’si :

- Best Case : O(n)
- Worst Case : O(n²)
- Average Case : O(n²)
